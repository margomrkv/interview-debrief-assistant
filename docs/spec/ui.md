## Описание

Мне нужно сделать интерфейс для моего приложения (AR-модуль).
Общие требования [[md/spec.md]]
Пользователь должен загружать транскрипт и ему должны показываться список вопросов и ответов с оценками и выводом.

## Интерфейс

Система работает в два этапа: 
1) Splitter 
2) Scoring

После первого этапа появляется список вопросов и ответов. 
После второго этапа появляется оценка по каждому из трёх критериев (Factual Correctness, Focus, Clarity) + вывод.

Система должна работать асинхронно, как только появится результат, он должен отображаться на странице.

## Технические детали

1) UI должен работать с двумя бекендами (emulator из файлов) и (реальный backend c llm).                                                                                                                                                                          
Реальный backend будет вызывать llm для spliiter (1 запрос) и assessor (N зарпосов) в заданными промптами.
2) Аутентификация не нужна, так как доступ к ui будет только у меня.
