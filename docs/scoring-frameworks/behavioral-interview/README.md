# Behavioral Interview Research Pack

Last updated: 2026-05-05

## Что это

Эта папка собирает в одном месте источники и рабочие определения для оценки behavioral-интервью в проекте.

## STAR методика (коротко)

STAR = Situation, Task, Action, Result.

- Situation: контекст, где и что происходило.
- Task: какая была задача и ожидания.
- Action: что именно сделал кандидат лично.
- Result: чем закончилось, какой эффект и метрики.

Зачем это нужно:
- структурирует ответ;
- снижает "воду";
- упрощает оценку по единым правилам.

## Предложенные шкалы оценки (0-3)

Базовые шкалы для всех вопросов:
- Question Fit: насколько ответ попадает в вопрос.
- Clarity: насколько ответ понятный и структурный.
- Completeness: насколько закрыты ключевые части вопроса.
- Accuracy: фактическая/техническая корректность.
- Substance: конкретика и содержательность вместо "воды".

Behavioral-дополнения:
- Scope: масштаб задачи/контекста.
- Contribution: личный вклад кандидата.
- Impact: результат и его измеримость.
- Difficulty: сложность, ограничения, trade-offs.

Шкала 0-3:
- 0: провал/нет сигнала;
- 1: слабо;
- 2: достаточно;
- 3: сильно.

## Рекомендуемые поля для разметки Q&A

- question
- answer
- question_type (hard | soft | behavioral)
- topic_tag (например: SQL, Python, Communication, Leadership)
- question_fit
- clarity
- completeness
- accuracy
- substance
- scope (только behavioral)
- contribution (только behavioral)
- impact (только behavioral)
- difficulty (только behavioral)
- notes
