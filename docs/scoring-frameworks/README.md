# Scoring Frameworks

Last updated: 2026-05-05

Папка с рабочими критериями оценки по типам интервью.

## Разделы

- `behavioral-interview/` — STAR и behavioral-оценка.
- `fit-interview/` — первичный fit/HR-этап.
- `technical-interview/` — техинтервью (вопросы + задачи/кейсы).
- `system-design/` — отдельный framework для case/system design раунда.
- `manager-round/` — финальный fit с будущим руководителем.

## Примечание по источникам

Репозиторий `awesome-behavioral-interviews` в основном покрывает behavioral-часть.
Критерии для technical/system design/fit/manager-round в этой папке собраны из методички Бориса.
