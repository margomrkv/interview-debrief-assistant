# Manager Round Scoring Notes

Last updated: 2026-05-05

## Что удалось найти в источниках

В методичке этот этап описан как отдельный важный раунд взаимного fit.
Формальной шкалы нет, но есть понятные критерии:

- совместимость стиля работы кандидата и будущего руководителя;
- прозрачность ожиданий по роли и росту;
- качество взаимной коммуникации;
- риск "токсичного mismatch" по команде/менеджеру.

## Практический критерий для вашей разметки (v0)

0-3 шкала:

- `manager_fit` — совпадение рабочих стилей;
- `expectation_alignment` — ясность и совпадение ожиданий;
- `trust_signal` — ощущение открытого, конструктивного диалога;
- `risk_flags` — признаки потенциальных конфликтов.
