# Technical Interview Scoring Notes

Last updated: 2026-05-05

## Что удалось найти в источниках

В методичке есть не "официальная шкала", а структура и наблюдаемые критерии:

- глубина технического опыта (что делал и почему принимал решения);
- базовые знания по стеку из вакансии;
- умение решать задачу/кейс с рассуждением вслух;
- качество процесса решения (уточнения, тест-кейсы, проверка кода).

## Практический критерий для вашей разметки (v0)

0-3 шкала:

- `tech_depth` — глубина и осознанность технических решений;
- `problem_solving_process` — структура рассуждения и проверка гипотез;
- `solution_quality` — корректность и адекватность решения;
- `communication_under_pressure` — ясность мысли в стрессовой задаче.
