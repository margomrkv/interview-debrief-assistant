# Fit Interview Scoring Notes

Last updated: 2026-05-05

## Что удалось найти в источниках

В материалах нет жесткой универсальной шкалы баллов для fit-интервью, но есть четкие сигналы, что оценивают:

- совпадение ожиданий кандидата и компании;
- ясность карьерных целей кандидата;
- мотивация к роли/компании;
- адекватность коммуникации и деловой тон.

## Практический критерий для вашей разметки (v0)

Можно использовать 0-3 шкалу по четырем осям:

- `goal_alignment` — совпадение ожиданий сторон;
- `motivation_clarity` — насколько внятно кандидат объясняет "почему эта роль";
- `communication_quality` — ясность и уважительность общения;
- `risk_flags` — красные флаги (негатив, противоречивость, неясные ожидания).
